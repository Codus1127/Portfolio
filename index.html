My First Project

And One Analytics

A user friendly, full-stack application that serves high-school and prep league basketball coaches an easy to use and persistent stat tracking platform. The goal: replace old-school stat books with a digital platform that could be easily used through a tablet on the sideline.
Users undergo a secure registration process and experience a private dashboard that allows them to add teams, players, and stats. Data visualization is done utilizing Chart.js to allow for easy trend identification.

React | Redux | HTML5 | CSS3 | NodeJS | Axios | Express | Express Session | Massive | PostgreSQL | Bcrypt | Chart.js

Next up...

U-Tube

You might have guessed: a new and improved media sharing platform! My team set out to create a video sharing service like Youtube but without many of the frills that can make Youtube obnoxious. A user can view, like, and comment on videos as well as create an account through which they can upload and manage their own content.
Videos are stored using Amazon S3 file storage and a smooth and secure login/registration is facilitated by the use of 0Auth. Team collaboration and version control was accomplished using GitHub.

React | HTML5 | CSS3 | NodeJS | Axios | Express | Massive | Heroku | PostgreSQL | 0Auth | Amazon S3 | Dropzone
