About Me

"I'm not an inspirational quote kind of person." - Chrissy Teigen

My love for code started in 2014 when one of my classes introduced me to HTML/CSS, SQL, and Excel VBA. My First Project

And One Analytics

A user friendly, full-stack application that serves high-school and prep league basketball coaches an easy to use and persistent stat tracking platform. Users undergo a secure registration process and experience a private dashboard that allows them to add teams, players, and stats. Data visualization is done utilizing Chart.js to allow for easy trend identification.

React | Redux | HTML5 | CSS3 | NodeJS | Axios | Express | Express Session | Massive | PostgreSQL | Bcrypt | Chart.js
